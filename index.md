## 📂 Understanding .lrc Files

An .lrc file is a simple text file that contains time tags. These tags tell your music player exactly when to show each line of text. When you name the .lrc file the same as your audio file, most players detect the lyrics automatically. This ensures a clean experience as you listen to your tracks.
